## Instalación y Uso

### Requisitos

- Unity 2022.3.62f1 (LTS)
- Android 7.0+ (API 24+) con ARCore compatible
- AR Foundation 5.1.x
- ARTrackedImageManager + AR Foundation

### Clonar el repositorio

```bash
git clone https://github.com/RubenCarty/rva-g04-tutor-anatomia-ra.git
cd rva-g04-tutor-anatomia-ra
```

### Abrir en Unity

1. Abrir Unity Hub
2. "Add" → seleccionar la carpeta del proyecto
3. Abrir con Unity 2022.3.62f1
4. `File → Build Settings → Android → Build and Run`
