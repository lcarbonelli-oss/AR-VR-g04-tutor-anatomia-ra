# Checklist de Avance Semanal — Tutor de Anatomía en RA

## Semana 01-02: Investigación y planteamiento

- [ ] Documento de investigación de mercado (apps similares)
- [ ] Definición del problema y usuario objetivo
- [ ] Primera idea del proyecto presentada al docente
- [ ] Repositorio creado y todos los miembros con acceso
- **Evidencia:** commits con documentos de investigación

## Semana 03-04: Prototipo conceptual

- [ ] MVP definido (lista de funcionalidades mínimas)
- [ ] Arquitectura técnica inicial
- [ ] Primer prototipo en papel (wireframes) o mockup
- [ ] Proyecto Unity creado y configurado para AR
- **Evidencia:** carpeta `docs/mockups/` con imágenes

## Semana 05-06: Primera iteración

- [ ] Escena Unity básica funcionando
- [ ] Al menos 1 funcionalidad AR implementada
- [ ] Marcadores o planos configurados
- **Evidencia:** video de 30 segundos de la primera versión

## Semana 07-08: Interacción

- [ ] Gestos táctiles implementados
- [ ] UI/UX básica implementada
- [ ] App funciona en celular Android
- **Evidencia:** APK de la versión beta instalado y funcionando

## Semana 09-10: Escenarios y UI

- [ ] Escenario 3D del proyecto finalizado
- [ ] UI completa con todos los elementos
- [ ] Feedback visual y sonoro implementado
- **Evidencia:** capturas de la escena en Unity

## Semana 11-12: Features especiales

- [ ] Feature específica del proyecto implementada
- [ ] (Si aplica) Datos de API integrados
- [ ] (Si aplica) Audio espacial configurado
- **Evidencia:** video de 1 minuto con la feature funcionando

## Semana 13-14: Calidad

- [ ] Pruebas de usabilidad con 2+ usuarios
- [ ] SUS calculado: _____
- [ ] Optimización: FPS medido: _____
- [ ] Al menos 3 correcciones basadas en usabilidad
- **Evidencia:** doc de usabilidad actualizado

## Semana 15-16: Entrega final

- [ ] Presentación de 10 min preparada
- [ ] README final actualizado
- [ ] APK final funcionando
- [ ] Video demo de 2-3 min
- [ ] Reflexión individual completada
- **Evidencia:** todo en el repositorio GitHub
